                                    <h2>砀山酥梨</h2>                                   
                                </div>
                                <p >解析梨的遗传密码，促进梨品种改良及产业发展，培育更好吃又优质的梨，丰富果篮子，满足市场需求，增加梨农收益，助力乡村振兴。</p>
								<p >外观特点：果实近圆柱形或不正广卵圆形，顶部平截稍宽，平均单果重 250 克，大果可达 1000 克以上。果皮为绿黄色，贮后变为黄色，果点小而密，果面不平，果皮薄。果心较小，果肉白色。

<p >
口感特点：砀山酥梨以酥脆甘甜著称，果肉洁白如玉，酥脆爽口，浓甜如蜜，汁水丰富，弹指即破，入口即酥，落地无渣，自古就有 “砀山梨，皮儿薄，落到地上找不着” 的说法。
<p >营养特点：砀山酥梨营养丰富，含可溶性固形物 11%-14%，可溶性糖 7.35%，可滴定酸 0.10%，维生素 C 含量 0.21 毫克 / 百克，还含有多种人体必需的氨基酸、矿物质等，具有较高的营养价值。
<p >生长特点：树势强，萌芽率为 82%，一般剪口下多抽生 2 个长枝。定植后 3 至 4 年开始结果，以短果枝结果为主，腋花芽结果能力强，丰产性好，管理好可实现丰产、稳产。
<p >品种特点：主要品种有金盖酥、白皮酥、青皮酥及伏酥等，以金盖酥品质最佳。不同品种在外观、成熟时间等方面略有差异，如白皮酥成熟较早，青皮酥成熟较迟，伏酥果形较大且成熟最早。

        async function checkTokenStatus() {
            try {
                const token = sessionStorage.getItem('token');
                if (!token) {
                    handleLogout();
                    return;
                }

                const response = await fetch('http://120.26.84.167:9111/tea/checkToken', {
                    method: 'POST',
                    headers: {
                        'Authorization': `Bearer ${token}`
                    }
                });

                if (!response.ok) {

                }

                console.error('Token检查失败:', error);
                handleLogout();
            }

        function updateUserActivity() {
            clearTimeout(sessionTimer);
            sessionTimer = setTimeout(handleLogout, SESSION_TIMEOUT);
        }

        function handleLogout() {
            const token = sessionStorage.getItem('token');
            if (token) {
                fetch('http://120.26.84.167:9111/tea/logout', {
                    method: 'POST',
                    headers: {
                        'Content-Type': 'application/json',
                        'Authorization': `Bearer ${token}`

                document.getElementById('login-dialog').style.display = 'none';
                document.getElementById('user-info').style.display = 'block';
                document.getElementById('username-display').textContent = username;
            }
        });

        function handleLoginButtonClick() {
            if (isLoggedIn) {
                handleLogout();
            } else {
                document.getElementById('login-dialog').style.display = 'block';
            }
        }
